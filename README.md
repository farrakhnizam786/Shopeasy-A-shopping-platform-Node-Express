## 📌 Overview

**Shopeasy** is a powerful and scalable **shopping platform** designed to deliver a seamless e-commerce experience.  
It supports **multi-role authentication (Customer, Admin, Super Admin)** and provides full control over users, products, and system operations.

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository
```bash
git clone https://github.com/farrakhnizam786/Shopeasy-A-shopping-platform-Node-Express.git
cd Shopeasy-A-shopping-platform-Node-Express
2️⃣ Install Dependencies
npm install
3️⃣ Configure Environment

Create a .env file:

PORT=5000
MONGO_URI=your_mongodb_connection_string
SESSION_SECRET=your_secret_key
4️⃣ Run Application
npm start

Development Mode:

nodemon server.js
